## Дизайн

Эстетика «пригласительного билета» по мотивам myata.com/invite (слёт-2025):
чёрно-белые секции с фигурными краями билета и пунктирными линиями,
центрированная колонка, крупный гротеск (Montserrat), золотые акценты,
рисованный от руки золотой круг на дате календаря, официальные логотипы
Lounge + Platinum, тиснёный орнамент на чёрном. Крупица AI-вайба:
«дышащее» свечение в герое и лёгкий ирисцентный отлив «В БУДУЩЕЕ».

- Текст и структура — 1-в-1 из оригинального ПДФ приглашения
- Форма заявки (имя / локация / телефон / гости) → Google Таблица
  «заявки на слёт» через тот же Apps Script веб-хук, что и Проект1
- Чистая статика: HTML + CSS + JS, без сборки и без 3D
